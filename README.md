# Python Arithmetic Operations

This repository contains a simple Python script demonstrating basic arithmetic operations in python.It allows users to perform calculations interactively by entering two numbers and selecting an operator.

## Operations Covered

* Addition
* Subtraction
* Multiplication
* Division
* Floor Division
* Modulus
* Exponent

  
Features
-Handles division by zero gracefully
-Provides clear prompts and instructions for users
-Easy-to-read, well-commented code
